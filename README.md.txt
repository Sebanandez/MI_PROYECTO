# TCG Vault - Mi Proyecto de Cartas

Esta es la página de mi proyecto para el curso. La idea es armar una tienda online para coleccionistas de cartas TCG, enfocada en todo lo que tiene que ver con los juegos de cartas, tales como Pokémon, Magic, Yu-Gi-Oh!, one piece, etc.

## 📦 ¿Qué tiene el proyecto hasta ahora?

Organicé todo en carpetas limpias para que no sea un problema para mas adelante.
En la barra de nav, cuando se pasa el mouse por "Productos", se abre un menú flotante para elegir el juego que querés buscar.
Separé las cartas por categorías (Pokémon, Magic, Yu-Gi-Oh!) usando Flexbox para que se acomoden solas.
Armé una sección de opiniones de clientes ordenada en tres columnas usando Grid.
Creé la página aparte con el formulario listo para conectarlo a Formspree y recibir los mensajes.
